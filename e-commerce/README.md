# E-Commerce Shop

A simple online shop with product listings, category filters, and a sliding cart drawer.

## What's Included

- **Product grid** showing items with emoji images, prices, and an "Add to Cart" button
- **Category filters** — click to show only clothing, accessories, or stickers
- **Cart drawer** that slides in from the right with item list, total, and checkout
- Warm, natural colour scheme

## Try It

Open `index.html` in your browser, or run:

```bash
cd e-commerce && python3 -m http.server 5500
```

Then visit `http://localhost:5500`.

## Ideas to Make It Yours

- Change the products and prices in `js/main.js`
- Replace the emoji with real product images
- Add a quantity selector so people can buy more than one
- Show a "sale" badge on discounted items
- Save the cart to localStorage so it survives page refreshes
