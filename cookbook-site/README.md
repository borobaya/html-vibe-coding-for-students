# Cookbook Site

A recipe collection you can browse, search, and filter by meal type.

## What's Included

- **Recipe cards** with emoji images, names, cook times, and tags
- **Tag filters** — show only breakfast, lunch, dinner, or snacks
- **Search bar** to find recipes by name
- **Recipe modal** — click a card to see full ingredients and step-by-step instructions
- Warm, kitchen-inspired design

## Try It

Open `index.html` in your browser, or run:

```bash
python3 server.py
```

Then visit `http://localhost:5500/cookbook-site`.

## Ideas to Make It Yours

- Add your favourite recipes to the list in `js/main.js`
- Replace emojis with real food photos
- Add a "Favourites" feature that saves liked recipes
- Let users submit new recipes through a form
- Add a "serves" count and difficulty rating
