# News Reader

A news website with a featured article, card grid, and category filters.

## What's Included

- **Featured article** displayed prominently at the top
- **Article cards** in a responsive grid
- **Category filters** — World, Tech, Sport, Science
- Clean, newspaper-inspired layout

## Try It

Open `index.html` in your browser, or run:

```bash
cd news-reader && python3 -m http.server 5500
```

Then visit `http://localhost:5500`.

## Ideas to Make It Yours

- Write your own articles in `js/main.js`
- Connect to a real news API to fetch live headlines
- Add a "breaking news" banner at the top
- Let users bookmark articles
- Add a dark mode toggle
