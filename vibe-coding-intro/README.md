# Intro to Vibe Coding

A 3-minute presentation introducing students to vibe coding with GitHub Copilot.

[🌐 Open Presentation](index.html)

---

## Slide 1 — What if you could build a website just by describing it?

No tutorials. No experience needed. Just your ideas.

![You describe → AI builds → You see it](assets/hero-flow.svg)

---

## Slide 2 — What is Vibe Coding?

| | Step | Description |
|---|---|---|
| <img src="assets/icon-describe.svg" width="32"> | 🗣️ **You describe** | Tell AI what you want in plain English |
| <img src="assets/icon-ai.svg" width="32"> | 🤖 **AI builds** | GitHub Copilot writes the code for you |
| <img src="assets/icon-browser.svg" width="32"> | 👀 **You see results** | Open the browser and watch it come to life |

It's like **directing a movie** — you decide *what* happens, the AI figures out *how*.

## Slide 4 — Here's what you'll do

|---|---|---|

## Slide 5 — Quick tips before you start

| | Tip | Description |
|---|---|---|
| <img src="assets/icon-target.svg" width="32"> | 🎯 **Be specific** | "Add a red button that plays a drum sound" works way better than "make it cool" |
| <img src="assets/icon-blocks.svg" width="32"> | 🧱 **Build step by step** | Start simple, then keep adding. Don't try to describe everything at once. |
| <img src="assets/icon-flask.svg" width="32"> | 🧪 **Experiment!** | Nothing can break permanently. Just try things and see what happens. |

Open VS Code, pick an idea, and tell Copilot what to build. **Go!** 🚀
