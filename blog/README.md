# Blog

A personal blog with posts, tags, search, and a sidebar.

## What's Included

- **Blog post cards** loaded from a simple list in JavaScript
- **Tag filtering** — click a tag in the nav to filter posts
- **Search bar** — find posts by title or content
- Warm, readable serif design

## Try It

Open `index.html` in your browser, or run:

```bash
python3 server.py
```

Then visit `http://localhost:5500/blog`.

## Ideas to Make It Yours

- Add your own posts to the list in `js/main.js`
- Create new tags for your interests
- Add a "Read more" page that shows the full post
- Try a dark mode toggle
- Add a "Most Popular" widget to the sidebar
