# Portfolio

A personal portfolio website to introduce yourself and show off your projects.

## What's Included

- **Hero section** with your name and a tagline
- **About Me** section you can customise
- **Projects grid** that loads from an easy-to-edit list in JavaScript
- **Contact form** with a thank-you message
- Dark theme with smooth hover effects

## Try It

Open `index.html` in your browser, or run:

```bash
python3 server.py
```

Then visit `http://localhost:5500/portfolio`.

## Ideas to Make It Yours

- Change the name, tagline, and about text
- Add your own project links to the list in `js/main.js`
- Drop an `avatar.png` into an `assets/` folder to add your photo
- Try a different colour scheme by changing the CSS variables
- Add a "Skills" section with icons or progress bars
