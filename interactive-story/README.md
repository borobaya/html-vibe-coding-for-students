# Interactive Story

A "choose your adventure" game where clicking different choices takes you down different paths.

## What's Included

- **Branching story** with 10 scenes — explore a mysterious castle!
- **Multiple endings** depending on the choices you make
- Smooth fade-in animations between scenes
- Dark, atmospheric theme

## Try It

Open `index.html` in your browser, or run:

```bash
python3 server.py
```

Then visit `http://localhost:5500/interactive-story`.

## Ideas to Make It Yours

- Write your own story by editing the `story` object in `js/main.js`
- Add more scenes and branching paths
- Use different emoji or add background images for each scene
- Add a character name input at the start
- Track and display stats like bravery or wisdom
- Add sound effects when making choices
